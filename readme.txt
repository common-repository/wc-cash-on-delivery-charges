== Description ==

**Woocommerce plugin to apply charges on cash on delivery payment method using pincode ** using this plugin you will get the option to charge cash on delivery amount from customer using woocommerce. Also you can set fixed COD (Cash on Delivery) price for all location
or you can set COD charges based on customer billing / Shipping details.

**Pincode and COD amount Importer ** function will provide you the option to import bullk pincode and COD amount data using CSV file format. You will also get option to add pincode and cod amount manually one by one.

**Cash on Delivery Amount ** You can add cash on delivery (COD) amount on each pincode / Zipcode separately. COD amount will display on checkout page under order details, if you selected the Cash on Delivery Payment Method.


==  **Plugin Features**

> * Enable/Disable COD
> * Change COD Term like fixed price for all location or COD price based on customer billing/ shipping details
> * Option to add fixed price COD amount for all location
> * CSV to import Pincode & COD Amount
> * Add COD Amount on each pincode
> * Option to add pincode and COD amount data manually
> * Change the display COD text label on checkout page


<h3>How to Setup Cash on Delivery Charges for WooCommerce</h3>


== Installation ==

1. Upload the directory `/wc-cash-on-delivery-charges/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "COD Price" menu and import bulk COD price and manage the setting
